# Movie Explorer

A simple movie discovery website where users can browse, search, and save their favorite movies.

##  Features

-  Search for movies
-  View movies in a card layout
-  Add/remove movies from favorites

##  Technologies Used

- HTML
- CSS
- JavaScript
